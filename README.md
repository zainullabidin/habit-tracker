# Habinator - Your Habit Tracker

A simple and fun habit tracker web app built with vanilla HTML, CSS and JavaScript. Track your daily habits, check them off on a weekly grid, and watch your streaks grow.

## Live Demo

https://zainullabidin.github.io/habit-tracker/

## Features

- Add and delete habits
- Weekly grid with toggleable checkmarks
- Today's column is highlighted
- Streak counter for each habit
- Navigate between weeks (previous, next, back to today)
- Data persists after page reload using localStorage
- Fun character Mr. Habinator to keep you motivated

## How To Run

```bash
git clone https://github.com/zainullabidin/habit-tracker.git
cd habit-tracker
open index.html
```

No dependencies. No build tools. Just open in a browser.
